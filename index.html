Alex Johnson Comp 20 Portfolio

This page contains all of my published work done in Tufts' Comp 20 class (spring 2017). The material consists of introductory web technologies, such as HTML, CSS, and JavaScript, as well as one server-side assignment. All work is divided up between smaller Labs and larger Assignments (at the moment this page only shows the labs).

Labs

Responsive

This is an example of responsive design as well as using spritesheets to limit the number of HTTP requests. Start with a large browser (greater than 1401 pixels), and decrease the size to see the page responsiveness.

https://tuftsdev.github.io/comp20-ajohnson/responsive/

Annotation (DOM)

To help Mariah Carey sing, we were tasked with annotating her 2017 New Year's Eve performance by writing JavaScript to annotate the video with the proper lyrics. Play the video and toggle the controls to see the lyrics change! (My Professor, Ming Chow, has a very good sense of humor!)

https://tuftsdev.github.io/comp20-ajohnson/annotation/

Messages

Since pure DOM manipulations can get quite verbose, this lab relies on JQuery to get JSON data (messages) from a herokuapp server and displays those messages on the page. (Since this page relies on a herokuapp, the first request might take a few seconds!)

https://tuftsdev.github.io/comp20-ajohnson/messages/
